## 先说说这玩意儿是干嘛的

WAATR是一家美国佛罗里达的公司,专门做智能净水杯。他们家的产品核心就一个理念:让你在任何地方都能喝上干净的水,而且不用依赖瓶装水。

他们的CEO是个博士,叫Rakesh Guduru。这哥们儿当年在旅行时喝了瓶装水,结果还是病了。从那之后他就琢磨,既然瓶装水也不一定安全,为什么不自己搞个能随时净化水的东西?于是就有了WAATR。

听起来挺简单的故事,但背后的技术还真不简单。

## 他们家的产品线

WAATR主要有两大系列:

**PureMax 4D** - 这是旗舰产品,获得过2023年红点设计奖和德国创新奖。它不光有UV-C紫外线杀菌,还内置了滤芯系统,所以叫"4D净化"。

**CrazyCap系列** - 这个更简单一些,主要就是UV-C杀菌功能,没有滤芯。但价格也更亲民,而且瓶盖可以装在其他品牌的水杯上,比如Hydro Flask、S'well这些。

## PureMax 4D到底有什么特别的

这款杯子我研究了挺久,确实有些独特的地方。

首先是**双UV-C LED灯**。一般的UV杀菌杯只有一个灯,PureMax用了两个20mW的LED,而且是在瓶盖里。你拧上盖子,按一下,紫外线就开始工作,3分钟左右能杀死99.9999%的细菌和病毒。

然后是**滤芯系统**。他们用的是椰壳活性炭加离子交换树脂的组合,能过滤掉重金属、化学物质这些东西。滤芯有三种选择:

- **Purist滤芯** - 基础款,主要过滤铅、汞、氯这些常见污染物
- **Artesian滤芯** - 除了过滤,还会添加矿物质,让水喝起来像泉水
- **Alkaline滤芯** - 提高水的pH值,偏碱性

一个滤芯能用大概60天,过滤120升水左右。

还有个挺有意思的功能:**自动清洁**。每小时杯子会自己用UV-C灯清洁一次内壁,所以不会有那种放久了发霉发臭的问题。

杯子本身是304不锈钢材质,三层保温设计,冰水能保持冷12小时,热水保温6小时。有25盎司(约750ml)、32盎司(约950ml)和40盎司(约1200ml)三种容量。

充电也挺方便,用的是磁吸式无线充电,4小时充满,能用一周左右。

## CrazyCap系列适合什么人

如果觉得PureMax太贵,CrazyCap系列是个不错的选择。

CrazyCap Pro用的是20mW的单UV-C LED,虽然没有滤芯,但杀菌能力跟PureMax是一样的。它有两种模式:

- **Normal模式** - 1分钟快速净化,适合自来水或饮水机的水
- **Crazy模式** - 3分钟深度净化,可以净化湖水、河水这种野外水源

最妙的是,CrazyCap的瓶盖可以装在其他品牌的水杯上。如果你已经有S'well或者Hydro Flask这种可乐瓶型的水杯,直接买个瓶盖就行,不用整个杯子都换。

价格方面,CrazyCap Pro 17oz大概99美元,25oz是119美元。CrazyCap 2.0更便宜,74美元左右。

## 真的有用吗?用户怎么说

我在网上看了不少评测和用户反馈,大部分人对净化效果是认可的。有个用户说他在野外徒步时用CrazyCap净化溪水,喝了几天也没拉肚子。

不过也有人提到一些小问题:

1. **喝水需要倾斜杯子** - PureMax的吸嘴设计让你得把杯子倾斜一定角度才能喝到水,不能像吸管杯那样直接吸
2. **偶尔有吱吱声** - 喝水时杯盖可能会发出轻微的吱吱声
3. **滤芯没有更换提醒** - 需要自己记着60天左右换一次滤芯

还有人在Trustpilot上给了差评,主要是针对之前众筹项目的配送问题。不过那都是2022-2023年的事了,现在官网直接购买的话,发货还是比较正常的。

## 跟普通水杯比,这个值吗?

这个问题我想了很久。

如果你:
- 经常出差、旅行,担心外面的水不干净
- 喜欢户外运动,徒步、露营什么的
- 对水质比较敏感,或者住的地方水质一般
- 想减少瓶装水的消费,更环保一些

那WAATR确实能解决你的问题。一年下来,不买瓶装水省的钱可能就够回本了。

但如果你:
- 大部分时间在家或公司,用的都是净水器的水

那可能就没必要花这个钱了。一个保温杯也就几十块钱,够用了。

## 有没有更便宜的替代品?

市面上确实还有其他UV净化杯,比如国内的一些品牌。不过很多都是单UV-C LED,没有滤芯系统,净化效果跟CrazyCap差不多,但做工和设计可能没那么精致。

如果你只是想要基础的杀菌功能,其实59刀的LARQ或者国内的一些品牌也能满足需求。但如果要UV加滤芯的组合净化,目前PureMax 4D确实没什么直接竞品。
